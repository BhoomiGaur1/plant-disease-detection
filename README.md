

# 🌿 Plant Disease Detection using CNN

## 📌 Project Overview

Plant Disease Detection is a deep learning based project that identifies plant leaf diseases from images.
The system uses **Convolutional Neural Networks (CNN)** along with **TensorFlow, OpenCV, and Python** to analyze leaf images and classify them into different disease categories.

Early detection of plant diseases helps farmers take timely action and improve crop productivity.

---

## 🚀 Features

* Detects plant leaf diseases using image classification
* Uses Convolutional Neural Network (CNN) for accurate predictions
* Image preprocessing and segmentation using OpenCV
* Supports multiple disease classes
* Jupyter Notebook implementation for easy experimentation

---

## 🛠️ Tech Stack

* **Programming Language:** Python
* **Libraries:** TensorFlow, OpenCV, NumPy, Matplotlib
* **Deep Learning:** Convolutional Neural Network (CNN)
* **Environment:** Jupyter Notebook

## ⚙️ Installation

Clone the repository:

```
git clone https://github.com/BhoomiGaur1/plant-disease-detection.git
```

Move to the project folder:

```
cd plant-disease-detection
```

Install required libraries:

```
pip install -r requirements.txt
```

---

## ▶️ How to Run

1. Open **Jupyter Notebook**
2. Open the file:

```
PlantShieldNet_Project.ipynb
```

3. Run all cells to train and test the model.
